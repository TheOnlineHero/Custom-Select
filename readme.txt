Stylise your select boxes.

== Description ==

Allows you to stylise your select boxes and makes them look more appealing. For example, you can change the down arrow of the select box.

== Installation ==

1) Install WordPress 3.5.2 or higher

2) Download the following files:
http://downloads.wordpress.org/plugin/tom-m8te.1.4.2.zip
http://downloads.wordpress.org/plugin/custom-select.1.0.zip

3) Login to WordPress admin, click on Plugins / Add New / Upload, then upload the zip file you just downloaded.
